Privacy Policy

About Colorize - Blind Test

Colorize - Blind Test is an application designed to conduct color blindness tests using the Ishihara method.

Data Collection

Colorize - Blind Test does not collect any personal data from its users. The application only requires internet permission to download test data and does not store or transmit any user-specific information.

Information Usage

The app solely uses internet permission to retrieve test data and does not utilize any data for purposes other than conducting color blindness tests.
